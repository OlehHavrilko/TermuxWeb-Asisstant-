# Termux Web Assistant

Многофункциональное приложение и веб-интерфейс для управления Termux на Android-устройствах. Позволяет управлять пакетами, файлами, выполнять скрипты и мониторить систему через удобный графический интерфейс.

Проект состоит из двух основных частей:
1. **Мобильное приложение (Flutter)** — для нативного управления на самом устройстве.
2. **Веб-версия (React + Node.js)** — для удаленного управления через браузер.

## 🚀 Возможности

- 💻 **Терминал** — Полноценный эмулятор терминала с поддержкой bash/zsh.
- 📦 **Пакеты** — Удобный менеджер пакетов `pkg`/`apt` (поиск, установка, обновление, удаление).
- 📁 **Файловый менеджер** — Удобная навигация, просмотр и редактирование файлов в среде Termux.
- 📜 **Скрипты** — Создание, редактирование и быстрый запуск bash/python скриптов.
- 📊 **Мониторинг** — Отслеживание нагрузки CPU, RAM и состояния памяти в реальном времени.
- ⚙️ **Настройки** — Кастомизация интерфейса и поведения приложения.

## 🛠 Технологии

### Мобильное приложение
- **Фреймворк:** Flutter 3+
- **Язык:** Dart
- **Архитектура:** Feature-first

### Веб и Сервер
- **Фронтенд:** React, Vite, Tailwind CSS, shadcn/ui
- **Бэкенд:** Node.js, tRPC
- **База данных:** Drizzle ORM (PostgreSQL / SQLite)

## 📦 Установка и запуск

### Мобильное приложение (Flutter)

Для сборки и запуска мобильного приложения требуется Flutter SDK.

```bash
# Установка зависимостей
flutter pub get

# Запуск в режиме отладки
flutter run

# Сборка APK для релиза
flutter build apk --release
```

### Веб-версия (React + Node.js)

Требуется Node.js 18+ и менеджер пакетов (npm, pnpm или yarn).

```bash
# Установка зависимостей
npm install
# или
pnpm install

# Запуск в режиме разработки (клиент + сервер)
npm run dev

# Сборка для продакшена
npm run build
```

## 📂 Структура проекта

```text
.
├── android/                 # Android конфигурация Flutter
├── client/                  # Фронтенд (React, Vite, Tailwind)
│   ├── src/                 # Исходный код клиента
│   └── public/              # Статические ассеты
├── drizzle/                 # Схемы и миграции базы данных
├── lib/                     # Исходный код Flutter приложения
│   ├── core/                # Ядро, сервисы и общие утилиты
│   └── features/            # Функциональные модули (terminal, files, и т.д.)
├── server/                  # Бэкенд (Node.js, tRPC)
│   └── _core/               # Базовые сервисы сервера
└── shared/                  # Общие типы и константы для web/server
```
